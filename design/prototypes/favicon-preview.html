🔷 Sapphire Mall - Favicon 设计

为 Sapphire Mall Web3 商城设计的专属图标

📱 图标预览

64×64 (原始)
32×32 (标准)
16×16 (小尺寸)

🎨 设计说明

设计理念：

  • 🔷 蓝宝石主题：使用蓝色渐变体现品牌特色
  • 💎 宝石形状：立体宝石造型象征价值与品质
  • S 品牌标识：融入字母"S"强化品牌识别
  • ✨ 现代感：适配Web3科技风格
  • 🌟 装饰元素：星点装饰增加视觉层次

🔧 生成 favicon.ico 的方法

方法一：使用在线转换工具（推荐）

使用以下在线工具将 SVG 转换为 favicon.ico：

Real Favicon Generator Favicon.cc Convertio

方法二：使用命令行工具

如果你安装了 ImageMagick：

# 将 SVG 转换为多尺寸 ICO magick favicon.svg -background transparent -define icon:auto-resize=256,128,64,48,32,16 favicon.ico

方法三：在线 SVG 转 ICO

使用专门的 SVG 到 ICO 转换服务：

1. 上传 favicon.svg 到转换网站 2. 选择输出尺寸：16x16, 32x32, 48x48 3. 下载生成的 favicon.ico 文件 4. 将文件放置到网站根目录

📝 HTML 集成代码

将以下代码添加到 HTML 的 <head> 部分：

<!-- 基本 favicon --> <link rel="icon" type="image/x-icon" href="/favicon.ico"> <link rel="icon" type="image/svg+xml" href="/favicon.svg"> <!-- Apple Touch Icon --> <link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png"> <!-- 各种尺寸的 PNG --> <link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png"> <link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png"> <!-- Web App Manifest --> <link rel="manifest" href="/site.webmanifest">

📁 推荐的 Favicon 文件结构

/ ├── favicon.ico # 16x16, 32x32, 48x48 (多尺寸ICO) ├── favicon.svg # 矢量图标 (现代浏览器) ├── favicon-16x16.png # 16x16 PNG ├── favicon-32x32.png # 32x32 PNG ├── apple-touch-icon.png # 180x180 (iOS) └── site.webmanifest # Web App 清单
